# :fishing_pole_and_fish: Fishing System Guide
## :package: Installation
1. Import **FE** into Studio
2. Move all folders to their correct locations
3. Make sure everything is **ungrouped**
4. You're done :white_check_mark:
## :gear: Requirements
* RadonTech Weather System
* Access to DataStore *(might work without it, not fully tested)*
## :warning:  Note
This is a **__development build__**.

* Start building the UI

* Setup Fishes/Rods/Areas
